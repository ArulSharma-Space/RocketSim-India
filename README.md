# RocketSim-India

RocketSim-India is a beginner-friendly Python project that simulates basic rocket motion using thrust, gravity, and drag.

## Features
- Simulates altitude, velocity, thrust, and drag.
- Uses basic physics.
- Lets you enter rocket mass, thrust, and burn time.
- Good beginner project for learning Python and aerospace basics.

## How to Run

### 1. Install the dependencies
```bash
pip install numpy matplotlib
```

### 2. Run the project
```bash
python rocket_sim_india.py
```

## Example Input
- Rocket mass: `2.0`
- Thrust: `50.0`
- Burn time: `3.0`

## What It Shows
The program plots:
- Altitude vs time
- Velocity vs time
- Thrust, drag, and weight vs time
