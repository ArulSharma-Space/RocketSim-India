# RocketSim-India

A simple Python rocket simulator that models altitude, velocity, thrust, and drag.

## Features
- Simulates rocket motion using basic physics.
- Shows altitude, velocity, thrust, and drag graphs.
- Beginner-friendly project for learning and portfolio building.

## How to Run
1. Install Python packages:
   ```bash
   pip install numpy matplotlib
   ```
2. Run the file:
   ```bash
   python rocket_sim_india.py
   ```

## Example
- Rocket mass: `5.0`
- Thrust: `100.0`
- Burn time: `4.0`
